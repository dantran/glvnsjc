Privilege

 

Each login account has an assigned privilege, which determines the level of accessibilities.  It is assigned by any account with principal or higher privilege using the Teacher menu.

 

There are 5 privilege levels:  class, school, principal, community, and admin.

Class

 

This privilege is suitable to Teacher Aid with the following accessibilities:

 

  • View all teacher records of his/her school
  • Edit all student records of his/her class

School

 

This privilege is suitable to Teacher with the following accessibilities:

 

  • View all teacher records of his/her school
  • Edit all student records of his/her school

Principal

 

This privilege is suitable to principal or a delegate teacher with the following accessibilities.

 

  • Edit all student/teacher records of his/her school.

Community

 

This privilege is suitable to head master or a delegate teacher with the following accessibilities:

 

  • Edit all student/teacher records of all schools

Admin

This privilege is suitable to system admin or a delegate teacher with the following accessibilities:

 

  • Edit all student/teacher records of all schools
  • Edit web site configuration

Benefits

 

One of the best example for using UNICODE in Window 2000 and XP is you can create a Microsoft Word document, save it as HTML format, and load the file into your web server. Any one (include those using Mac, Windows 9x) can view the document without installing any Vietnamese font

 

However, in order to view a Vietnamese Word document, you must have Vietnamese language support in Window 2000/XP. There is no support for Windows 9x

 

Set up Windows 2000

 

  • Open the “Control Panel” window using StartàSettingsàControl Panel
  • In “Control Panel” window, invoke “Regional Options” icon to open “Regional Options” dialog
  • In the “Regional Options” dialog, put a check for “Vietnamese” in the “Language settings for the system” (you need to scroll down to the option). Hit OK and Window 2000 will install Vietnamese language support. Prepare to have Windows 2000 installation CD ready. Restart your PC if asked. This step installs the software.
  • In the Input Locale tab, add “VI” for Vietnamese support. This step activates Vietnamese support.

 

Set up Windows XP

 

  • Open the Regional and Language Option dialog using Start à Control Panel à Date, Time, Language, and Regional Optionsà Regional and Language Option
  • In the Regional and Language Option dialog’s Language tab, put a check for “Install files for complex script and right to left language…” and hit OK. Window XP will install a set of languages including Vietnamese. Prepare to have Windows XP installation CD ready. Restart your PC if asked.
  • In the Regional and Language Option dialog’s Language tab, hit “Details…” button to open “Text services and input language” dialog. In this dialog, hit “Add” button in “Installed Services” box to open “Add Input Language” dialog and select “Vietnamese” and “US” for “Input Language” and “Keyboard Layout…” respectively. Hit “OK”

Usage

 

  • After adding Vietnamese language support to your Windows system, you should see “EN” (English) in the lower right task bar. Use Left-ALT Shift to toggle between “EN” and “VI” (Vietnamese)
  • In “VI” mode, number 1 to ̣9 stand for ă â ê ô ̀ ̉ ̃ ́and ̣ . ‘0’, ‘[‘, and ‘]’ stand for đ ư and ơ .
  • Using normal shift key for capitalization.

Frequently Asked Questions (FAQ)

 

 

1.      Excel exported from student list shows gable student names

 

Currently, When you hit the Excel export link at the bottom of a student list, a popup dialog shows you 2 options: 1) Open the export directly by feeding the exported data to Microsoft Excel, or 2) Save the export data into a file on your local hard disk.  Option 1 shows incorrect display of Unicode (Việt font) in the Excel spreadsheet.  The worked around steps are:

 

·        Save the export file to your local hard disk

·        Use notepad to open the export file and copy every thing to the clipboard (i.e. hit control-A to select every thing, then control-C to copy)

·        Close the export file.

·        Open a blank Excel file and do a paste (Control-P)

3.      Why student search engine does not support Unicode search?

 

Yes it does.  If you input all your students information manually using the add feature (only Master account can do this), then the search will work. 

 

However the data originally was in VNI format, I converted it to Unicode using VNI to Unicode conversion tool before imported to database. 

 

Here is the interesting fact. The Unicode data in the database and the Unicode you type on the search screen are not the same.  There are 2 ways to represent Nguyễn in Unicode format.  The standard Unicode way and the Microsoft way.  This messes the search engine. 

 

The solution is to convert the standard Unicode to Microsoft Unicode. 

Any volunteer to research into this problem?

 

However, even if I can do the conversion I still recommend to use the wild card search (i.e Nguy% for Nguyễn) since Vietnamese has different ways to represent Viet words (Nguyễn vs Nguye^~n)

Student Search

 

The student search form contains the most frequently used criteria (fields) to find one or more students.  Multiple fields criteria acts as a logical “AND” operation to perform a find grain search.  Last statement sounds complicated, but it is not really!! It is best to provide some examples:

 

Note: The examples assume that you only enter the targeted fields and leave other editable fields blank

 

1.      Search for all students.

Leave all editable fields blank

 

2.      Search for student with specific student identification

Enter student ID into student ID filed.

 

3.      Search for all students with the same birthday.

Enter the birth date using strict format of MM//DD/YYYY (i.e. 01/01/2001)

 

4.      Search for all students with similar phone number

Enter wild card phone number ( i.e. %223%) in the phone field

 

5.      Search for all student with last name “Nguyẽn”

Enter “Nguy%” into Last Name field. Due to an unresolved Unicode bug, you cannot enter the full last name “Nguyẽn”

 

6.      Search for all students start with “Tr” like Trương and Trần

Enter “Tr%” into Last Name field.

 

7.      Search for  all students in Giáo Lý 3A.

Enter “3”  and “A” in the  first and second field of Giáo Lý  group respesctively.

This is multiple fields criteria

 

8.      Search all students in Giáo Lý 3 include all subclasses

Enter “3” in the first field of Giáo Ly ́ field groups

 

9.      Search all students who attend both Giaoly 5 and Vietngu 4.

Enter  “5” in the first field of Giaoly group and “4” in the first field of Vietngu group.  This is multiple fields search criteria.

 

Other notes:

 

  • Last Name, Middle Name, First Name, and Phone fields accept wide card character ́(%).  For example: Nguy%, %223%

 

  • Student ID is the prefer search criteria.  Otherwise use birth date, combination of last name and first name, or phone to lookup the student.
